# shrobana.github.io

This repository hosts a simple static website for GitHub Pages. The site includes:

- `index.html` — main homepage content
- `styles.css` — responsive styling
- `script.js` — mobile menu toggle

## How to use

1. Open `index.html` in your browser to preview locally.
2. Push this repository to GitHub.
3. If this repo is named `shrobana.github.io`, GitHub Pages will publish it automatically.

## Customize

- Update the hero section text in `index.html`
- Change colors and spacing in `styles.css`
- Add new sections or project cards as needed
